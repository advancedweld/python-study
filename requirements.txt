# 依赖说明： 

# Flask
# Flask 是一个轻量级的 Web 框架，用于构建后端服务。

# Flask-SQLAlchemy：
# Flask 的 SQLAlchemy 扩展，用于与数据库进行 ORM 映射。

# Flask-Migrate：
# 数据库迁移工具，基于 Alembic，用于管理数据库 schema 的变更。

# Flask-CORS：
# 处理跨域资源共享（CORS）的扩展，允许前端应用访问后端 API。

# Flask-JWT-Extended：
# 用于实现 JWT（JSON Web Token）认证的扩展，支持用户登录和权限管理。

# PyMySQL：
# MySQL 数据库的 Python 驱动，用于连接 MySQL 数据库。

# python-dotenv：
# 用于加载 .env 文件中的环境变量，方便管理配置。
